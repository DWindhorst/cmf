Detailed Description

integratable is a functionality for different classes for integrating
values over time.

Main usage of an integratable is the calculation of average fluxes
over time e.g. 

.. math::


    \\int_{t_0}^{t_{end}}q\\left(t,V_i,V_j\\right)dt
